What is Kengin?

Kengin is a data-driven game engine for creating large scale worlds.

Kengin uses the latest ideas in data driven processing to enable high performance virtual experiences.

The main coding language is the popular language Rust.

Create and publish games

It's easy to get started with Kengin.

Using Rust Cargo's build system, running your game can be as simple as cargo run.

Highly scaleable

We have a flexible architecture where parts of the system can scale independently which leads to lower server costs.

Build new worlds

Create large, responsive environments with our world building tools.

Worlds can be deployed locally or distributed over cloud and edge servers.

Why Rust?

Rust is high performance and largely memory safe. It supports many deployment targets including WASM (for web deployment).

Rust comes with a great package manager (Cargo) which makes it easier to buld and deploy projects. Also the Rust community is great!

When can I use Kengin?
